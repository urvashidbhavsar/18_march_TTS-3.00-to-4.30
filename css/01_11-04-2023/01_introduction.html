CSS => cascading stylesheet

        - for designing in website
        - make a responsive and flexible website.
        - set animation, set transitions, set transform etc.

        how many ways/type to create css :
        (1) inline = with set style attribute
        (2) internal = using style tag in < head > tag
        (3) external = create an external file and link with < link > tag.
                    file name like => style.css

        syntax :
        (1) inline 
            style = "property : value;"

        (2) internal / external 
            selector{
                property : value;
            }
        
        selector :-
        (1) element selector 
        (2) class selector 
        (3) id selector 
        (4) group selector 
        (5) universal / global selector
